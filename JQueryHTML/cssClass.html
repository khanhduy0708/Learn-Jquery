Phương thức thêm xóa CSS Class: 

        addClass(): thêm 1 hoặc nhiều class css vào thẻ được chọn
        removeClass(): xóa 1 hoặc nhiều class css của thẻ được chọn
        toggleClass(): chuyển đổi giữa thêm và xóa class css
        css(): đặt thuộc tính cho class css
            cú pháp: css("propertyname","value"); ví dụ: $("p").css("background-color", "yellow");
                     css({"propertyname":"value","propertyname":"value",...}); // nhiều thuộc tính
                     css("propertyname"); // return trả về thuộc tính
